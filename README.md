# davtestpy
WebDav testing tool

```
Usage: davtest.py [options]

Options:
  -h, --help            show this help message and exit
  -u URL, --url=URL     hostname
  -l USERNAME, --username=USERNAME
                        username
  -p PASSWORD, --password=PASSWORD
                        password
  --proxy=PROXY         proxy URL
  --clean               remove uploaded files
  --dont-test           do not upload test files
  --upload-file=UPLOAD_FILE
                        upload just specified file

```
